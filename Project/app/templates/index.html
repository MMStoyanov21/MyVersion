WELCOME

Aim of project

This project aims to develop a predictive model for residential house prices using a feed-forward artificial neural network. The model is trained on a structured dataset containing detailed property characteristics, architectural features, location information, and building quality indicators. Each row in the dataset represents a single residential property, while each column represents a feature used for predicting the sale price.

How to use the predictor

Upload a CSV file containing the required house-related information such as lot size, zoning classification, building type, number of rooms, basement details, garage size, quality ratings, and other property features. Once the file is uploaded, the predictor processes the input data using the trained neural network and returns an estimated sale price.

Description of the neural network

This neural network is feed-forward (FFNN). It has 128 neurons in the first layer, 64 in the second and in the last one it has 1. Every neuron form the first two layers has an activation function that is sigmoid. The training is done with 60 epoch and a batch size of 64, it uses an 80/20 split for minimal training loss.
